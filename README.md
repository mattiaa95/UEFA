# UEFA
UEFA Code Test UIKit, no Pods - no Swift package

## Requesites
* Swift 5.0
* Xcode 13.1

### Installation
* `git clone https://github.com/mattiaa95/UEFA.git`
* `cd /UEFA`
* `open UEFA.xcodeproj`

### App Tree
```
├── App
│   ├── AppDelegate.swift
│   ├── SceneDelegate.swift
│   ├── Main
│   └── ClubSquad 
└── Supporting Files
    ├── Assets.xcassets
    ├── Base.lproj
    └── Info.plist
```
